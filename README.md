### Prerequisites
- Node.js v18+
- Starknet wallet (ArgentX or Braavos)
- Yarn/NPM/PNPM/Bun

### Installation
1. Clone the repository:
```bash

```
2. Install dependencies :
```bash

# or
yarn
# or

```

# or
bun dev
